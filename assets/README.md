## Printing Tips

### Recommended Format
- Print on **card stock** (65 lb or heavier) for durability in the field.
- Size: Standard poker card size (2.5" × 3.5") fits most card sleeves.
- Print double-sided: card back on one side, card text on the other.

### Tools for Designing Cards
- **Canva** — Free templates shared by the fan community; search "Jet Lag card template."
- **Google Slides / PowerPoint** — Manually create cards using the descriptions below.
- **Inkscape / Illustrator** — For SVG-based custom designs.

### Laminating
For outdoor/vehicle use, laminate printed cards or sleeve them in clear card sleeves (standard MTG/poker size).
