## Contributing Challenges

Have a great rural challenge idea? Open a pull request to add it to [`challenges.md`](./challenges.md). Each challenge entry should include:

- **Name:** Short, memorable name for the card
- **Type:** Task / Navigation / Trivia / Social
- **Description:** What the player must do
- **Reward:** What the completing team receives
- **Difficulty:** Easy / Medium / Hard
- **Notes (optional):** Any regional variations or prerequisites

See [CONTRIBUTING.md](../CONTRIBUTING.md) for full guidelines.
